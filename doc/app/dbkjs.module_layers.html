Module: layers

A collection of functions to manipulate map layers and dialogs related to map layers

Methods

(inner) createBaseLayers()

Create the dialog for Base layers on the map that gives a user the option to toggle layers on and off. Will also add event listeners to the loadstart and loadend events to display layer loading progress
