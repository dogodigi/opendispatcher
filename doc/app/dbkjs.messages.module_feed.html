Module: feed

Grabbing information from the p2000 paging monitor. Particular for the Netherlands

The standard url for feeds is http://feeds.livep2000.nl The feed can be accessed with a GET and can be complemented with the parameters 'r' and 'd', like

?r=1,2&d=2

2 Friesland
3 Drenthe
